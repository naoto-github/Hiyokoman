# ひよこマン

enchant.jsを利用したアクション・ゲームです．
2014年に椙山女学園大学で開催された[ワークショップ・ギャザリング](http://workshop.ciao.jp/gathering/)において，
向研究室で「自分だけのオリジナル・キャラクターを創ってゲームしよう！」というタイトルで出展した作品の一つです．
